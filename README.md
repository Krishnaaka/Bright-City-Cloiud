# Simple Starter Project

Yes — you can absolutely create a simple project and share it on GitHub.

This repository now includes a tiny example file (`hello.py`) so you can see a complete basic setup.

## 1) Run the example locally

```bash
python3 hello.py
```

## 2) Initialize git (if needed)

```bash
git init
git add .
git commit -m "Initial commit"
```

## 3) Create a GitHub repository

1. Go to <https://github.com/new>
2. Choose a repository name.
3. Click **Create repository**.

## 4) Push your code to GitHub

Replace `YOUR_USERNAME` and `YOUR_REPO` with your values:

```bash
git remote add origin https://github.com/YOUR_USERNAME/YOUR_REPO.git
git branch -M main
git push -u origin main
```

Once pushed, share the repository URL with anyone.
